

## Overview
This project builds a pipeline combining image deblurring and object detection.

The goal is to analyse how restoring blurred images affects detection performance.

## Tasks
- Version control and AI-assisted development
- Image deblurring (motion blur removal)
- Evaluation using PSNR and SSIM
- Object detection using YOLO
- Dataset preparation and training
- Performance comparison

## Tools Used
- Python
- Jupyter Notebook
- OpenCV
- NumPy
- Matplotlib
- Ultralytics YOLO

## Project Structure
- notebooks/ → implementation
- src/ → helper functions
- data/ → dataset
- outputs/ → results
